# 🥗 Healthy Diet - 饮食健康顾问

AI 驱动的饮食健康顾问技能，提供科学的营养建议、膳食搭配和健康饮食指导。

## 📋 功能特性

- ✅ 完整的膳食结构参考（基于中国居民膳食宝塔 2022）
- ✅ 各类人群饮食方案（减重期、增肌期、控糖、素食、外卖族）
- ✅ 热量速查表（主食/蛋白质/蔬菜/水果/零食）
- ✅ 升糖指数（GI）参考表 + 降低整餐 GI 的实用技巧
- ✅ 营养素摄入建议
- ✅ 食物选择与搭配指导

## 🚀 安装方式

### 方式一：使用 ClawHub（推荐）

```bash
clawhub install healthy-diet
```

### 方式二：手动安装

```bash
cd ~/.openclaw/workspace/skills
git clone https://github.com/Dannyzn/healthy-diet.git
```

## 💡 使用方法

安装后，直接向 AI 助手提问即可自动激活此技能：

**触发词示例：**
- "吃什么比较健康？"
- "怎么搭配营养均衡的一餐？"
- "这个食物热量高吗？"
- "糖尿病患者怎么吃？"
- "孕妇需要补充什么营养？"
- "减肥期间可以吃什么？"

## 📚 包含内容

### 核心文档
- `SKILL.md` - 技能主文档，包含完整的饮食指导逻辑
- `references/calorie-table.md` - 常见食物热量速查表
- `references/glycemic-index.md` - 升糖指数（GI）参考表

### 适用场景
- 日常饮食咨询
- 特殊人群饮食建议（孕妇、老人、运动员等）
- 慢性病饮食管理（糖尿病、高血压等）
- 营养素补充建议
- 食谱推荐与优化
