# 常见食物升糖指数（GI）参考表

## GI 分级标准
- **低GI（≤55）**：血糖上升缓慢，适合控糖人群
- **中GI（56–69）**：适量食用
- **高GI（≥70）**：血糖上升快，控糖人群需限制

## 降低整餐GI的实用技巧

1. **搭配膳食纤维** — 蔬菜先吃，主食后吃，可降低血糖峰值
2. **增加蛋白质和脂肪** — 减缓碳水吸收速度
3. **选择完整食物** — 加工越少GI越低（糙米 < 白米 < 米粥）
4. **冷却淀粉** — 米饭、土豆冷却后产生抗性淀粉，GI降低
5. **醋/柠檬汁** — 酸性环境减缓淀粉消化，降低GI约20%
6. **细嚼慢咽** — 延长进食时间，减缓血糖上升
